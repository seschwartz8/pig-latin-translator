# Pig Latin Translator

#### Translates words and sentences into Pig Latin.

#### By Sarah "Sasa" Schwartz

## Description

This project was written with Javascript to translate any English word or sentences into Pig Latin. The user interface was written with HTML, CSS, Bootstrap, and jQuery.

### Specs
Program does nothing to non-alphabetical characters
* Example input: 3
* Example output: 3    
<br/> Program adds "way" to words beginning with a vowel
* Example input: in
* Example output: inway    
<br/> Program moves all consecutive consonants at start of word to end and adds "ay"
* Example input: Pig
* Example output: igPay    
<br/> Program treats "y" as a consonant for words that start with "y"
* Example input: yarn
* Example output: arnyay    
<br/> Program moves "u" characters that follow "q" from start of word to end as well
* Example input: squeal
* Example output: ealsquay    
<br/> Program works for sentence inputs as well as words
* Example input: I am a pig
* Example output: Iway amway away igpay

## Setup/Installation Requirements

* Download or clone repo to your computer
* Open the root directory
* Double click the index.html file to open it in a browser
